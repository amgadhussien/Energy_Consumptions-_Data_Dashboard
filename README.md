## 📌 Project Overview
This project provides a comprehensive analysis of energy consumption (Electricity, Water, and Gas) across 11 buildings. The goal was to build a robust system that calculates costs dynamically and tracks consumption efficiency over multiple years.

---

## 🏗️ Data Modeling (The Core Architecture)
The highlight of this project is the **Relational Data Model** created to handle complex utility calculations:
* **Schema Design:** I transformed separate raw files into a structured model by connecting the **Consumption Logs** (Fact) with the **Rates Table** and **Building Master** (Dimensions).
* **Dynamic Cost Calculation:** Instead of static numbers, I modeled the data so that costs are calculated based on the specific **Yearly Rate** for each energy type, allowing for accurate financial reporting.
* **Data Integration:** Merged geographic data with consumption metrics to enable location-based analysis.
